Welcome to my humble project!
The goal was to developp in less than 2 weeks a game similar to 1980 'Rogue', which created the roguelike genre.
In my game, you will fight monsters from rats to dragons, loot awesome gears and unlock power-ups to help you explore the dungeon.


HOW TO USE:

This game currently only works on linux.
In the main folder, use the command "make". Make sure you have the SFML graphical library installed.
Then, use the command "./rogue" to start the game.


Controls:

z-q-s-d to move
Moving on a tile occupied by an enemy will attack them.
For range attacks, use the arrow keys to select a tile, and if it is within range, press enter to damage the enemy.
Killing an enemy will give you exp and has a chance of dropping an item (gear or health potion).
Levelling up will allow you to choose a power and give you a skill point to be spent in the inventory.
Press tab or I to enter the inventory.
In the inventory, press 1-3 to use a skill point and the arrow key to select and equip gear.
The space in your inventory is limited, use backspace to discard a useless item.
There is currently no true end goal or victory condition.
